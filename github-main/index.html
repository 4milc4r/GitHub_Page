git init Crea un nuevo repositorio Git en el directorio actual.
git config --global user.name "usuario" Establece tu nombre para los commits.
git config --global user.email "correo@ejemplo.com" Establece tu dirección de correo electrónico para los commits.
git add "archivo" Agrega un archivo al índice para ser comprometido.
git commit -m "Mensaje descriptivo" Crea un nuevo commit con el mensaje especificado.
git status Muestra el estado actual del repositorio.
git branch "nombre_rama" Crea una nueva rama.
git checkout "nombre_rama" Cambia a la rama especificada.
git merge "rama" Combina los cambios de una rama en la actual.
git clone "url_del_repositorio" Clona un repositorio remoto a tu máquina local.
git remote add "nombre" "url" Agrega un nuevo repositorio remoto.
git push "origen" "rama" Envía los cambios locales al repositorio remoto.
git pull "origen" "rama" Descarga los cambios del repositorio remoto y los combina con los locales.
git log Muestra el historial de commits.
git diff Muestra las diferencias entre los archivos.
git reset --hard "commit" Deshace los cambios desde un commit específico.
git revert "commit" Revierte un commit.
git stash Guarda cambios no comprometidos temporalmente.
git stash pop Aplica los cambios guardados en el stash y elimina la entrada del stash.
git fetch Descarga objetos y referencias de un repositorio remoto sin aplicarlos.
git rebase "rama" Reaplica commits en una nueva base.
git tag "nombre_etiqueta" Crea una etiqueta (tag) en el commit actual.
git rm "archivo" Elimina un archivo del índice y del sistema de archivos.
